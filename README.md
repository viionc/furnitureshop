# Furniture Shop Demo: https://furnitureshop-ochre.vercel.app/

E-commerce app used to practice frontend development skills. 

# Technologies used:
- Vite,
- React,
- TypeScript,
- framer-motion,
- typed.js
- TailwindCSS

# To build the app type in terminal:
``` js
npm run build
```
